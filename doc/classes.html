GaMaLib

This is a working draft list of basic GaMaLib classes. The text is incomplete and comes with unstable version 1.1 of GaMaLib library. For more information about GaMaLib project of free geodetic software visit web pages at http://gama.fsv.cvut.cz/gama/ or contact authors.

Header file with definitons of PointData, ObservationData, PointIDList

Points

classdescription
Point

Coordinates and characteristics of point. Attributes of a point used in adjustment are:

fixed
Point with fixed given coordinates x and y
free
Point with known preliminary coordinates x and y which are beeing adjusted
constrained
Points with known preliminary coordinates x and y which are used for regularization of free network adjustment. In the case of free network adjustment additional condition must be be fulfilled: Sum (x2i + y2i) = min., for i=1, ..., n, where n is the number of constrained points.
PointData

Container class for points handling.

Observations

classdescription
Observation

Base observation class. All observations used in adjustment must be linked to a Cluster class; see the figure. Pointer from Observation object to Cluster can be ignored when calculating preliminary coordinates etc.
Distance

Derived form Observation - measured horizontal distance

Direction

Derived form Observation - measured horizontal direction
Angle

Derived form Observation - measured horizontal angle
X Y Z

Observed coordinates X, Y, Z (with given covariance matrix)
ObservationData

Container class for handling list of observations; see the figure.
Cluster

Container class for handling sets of observations. Cluster stores information common to a set of observations (namely orientatin for the set of observed directions or covariance of correlated observations).

StandPoint

Derived from Cluster. Container class for observed distances, directions and angles, possibly with common covariance matrix.

Coordinates

Derived from Cluster. Container class for observed coordinates X, Y and/or Z with common covariance matrix.

HeightDifferences

Derived from Cluster. Container class for observed height differences (levelling) with common covariance matrix.

Preliminary coordinates

classdescription
PriblSour

Class PriblSour calculetes aproximate coordinates of points based on input data from PointList and ObservationList. PriblSour works in iterations until no point without coordinates is left int the lists or until no more coordinates can be calculated from given data (last iteration produced no result).

Orientation

Class Orientation calculates aproximate orientation angle for a set of horizontal directions. The class can be used for computing of all missing orientations in PointList and ObservationList.

Adjustment

classdescription
NetInfo

Class NetInfo represents interface to geodetic 2D network adjustmen. Least Squares adjustment (of possibly free) geodetic network is based on Singular Value algorithm (SVD). Alternatively you can compile GaMalib library to use adjustment algorithm based on Gram-Schmidt ortogonalization (GSO).
